Ejercicio 10: Selectores combinados

Este párrafo se estiliza con selector descendiente: main p.

Este párrafo se estiliza con selector combinado etiqueta+clase: p.especial.

Este párrafo se estiliza con selector hijo directo + ID: .contenedor > p#unico.

Diferencia

Un selector combinado une condiciones o relaciones entre elementos. Por ejemplo, main p selecciona párrafos dentro de main, p.especial selecciona solo párrafos con esa clase y .contenedor > p#unico selecciona un párrafo hijo directo con ese ID.
