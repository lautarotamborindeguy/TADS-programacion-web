Ejercicio 1: Identificación de tags semánticas

<header>: representa la cabecera de una página o de una sección, normalmente contiene título, logo o introducción.

<nav>: contiene enlaces de navegación para moverse entre secciones o páginas.

<main>: agrupa el contenido principal y único de la página, es donde debemos colocar el contenido principal de nuestra web.

<footer>: representa el pie de página o de sección, con información final como autor, contacto o derechos.
